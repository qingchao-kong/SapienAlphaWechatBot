# SapienAlpha Wechat Bot

## 一、运行
0. 安装 Node.jS (>=10)
1. clone 代码
```text
git clone git@github.com:qingchao-kong/SapienAlphaWechatBot.git
cd wechatbot
```
2. 安装依赖
```text
npm install
```
3. 运行Bot
```text
npm run start:wechat:web
```
运行成功后，你可以看到下面的界面：
![img.png](source/img.png)
打开日志中的链接（日志中的二维码扫描识别速度较慢），扫码登录。
